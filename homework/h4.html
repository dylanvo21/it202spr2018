Answers to questions

  1. The box model is the idea of a box wrapping around each HTML element.
  2. It means an element that occupies the entire space of its parent element.
  3. Div is a block level element tag.
  4. Inline elements are those which only occupy the space bounded by the tags defining the element.
  5. The a tag is by default an inline element.
  6. One way CSS can be included is through an external style sheet. Another is through inlining.
  7. They are patterns used to select elements that you want to style.
  8. The text of the rule would simply be body{background-color:#0000ff;}.
